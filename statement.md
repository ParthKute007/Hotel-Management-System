# Restaurant Billing & Table Allocation System - Statement

## **Problem Statement**
Managing table allocation, taking customer orders, calculating bills, and freeing tables manually can be slow and error‑prone in busy restaurants. The aim of this project is to create a simple Python program that automates the table allotment process, handles customer orders, and generates a bill efficiently.

## **Scope of the Project**
- Automates table allocation for customers.
- Displays a menu with item prices.
- Allows customers to place multiple food orders.
- Generates a text bill with all ordered items and total amount.
- Frees the table once the bill is paid.
- Ensures smooth restaurant operations using basic Python features.

## **Target Users**
- Small restaurants and cafés.
- Beginners learning Python-based restaurant management systems.
- Students working on simple programming projects.
- Owners needing a basic billing and table management solution.

## **High-Level Features**
- **Table Allocation:** Automatically assigns the first available free table.
- **Menu Display:** Shows a predefined list of dishes with their prices.
- **Order Taking:** Allows customers to order multiple items with quantities.
- **Bill Generation:** Creates a detailed text file showing all items and total cost.
- **Table Release:** Marks the table as free after payment.
- **User-Friendly Flow:** Continuous loop allowing multiple customers until exit.
